## 📝 폴더 관리

본인의 이름 폴더를 만들고 > 주차별 폴더 > 코드 파일

[예시]

```
유환태
   ㄴ WEEK01
      - BOJ_1010

```

## 📕 Commit Convention

해당 컨벤션을 꼭 지켜주세요.
한 문제당 하나의 commit 부탁드립니다.

이름/플랫폼/문제번호/시간

[예시]

```
유환태/BOJ/1010/364ms
```
